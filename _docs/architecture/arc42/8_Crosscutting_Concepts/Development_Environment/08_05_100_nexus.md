# Setup

`docker-compose.yml`
~~~yml
nexus:
    image: sonatype/nexus3
    volumes:
      - "nexus3-data:/sonatype-work"
    ports:
      - "8081:8081"
volumes:
  nexus3-data:
~~~

Hinweis: Volumes befinden sich unter `/var/lib/docker/volumes`.
